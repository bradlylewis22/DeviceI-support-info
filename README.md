# DeviceI-support-info

This Chrome extension displays usefull information about your Chrome device to provide to your internal IT support. 

Information provided by this extension:
  Serial Number
  Asset ID: (Friendly Name in Google Admin Console)
  Hostname
  IP Address
  Mac Address
  Location (Assigned in Google Admin Console)

The Extension also provides a Copy all to clipboarb feature to allow end users to copy and paste their device information to their support team.
